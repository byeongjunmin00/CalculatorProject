## 주요 기능

- 두 개의 숫자 입력
  
- 사칙연산 기호 입력
  
- 덧셈, 뺄셈, 곱셈, 나눗셈 수행
  
- 나눗셈에서 0 입력 시 예외 처리
  
- 잘못된 연산자 입력 시 안내 메시지 출력
  
- 계산 결과를 ArrayList에 저장
  
- 저장된 결과 중 가장 먼저 저장된 값 삭제 가능

## 사용 기술

- Java
  
- Scanner
  
- ArrayList

## 실행 방법

1. 프로젝트를 실행합니다
   
2. 첫 번째 숫자를 입력합니다
   
3. 두 번쨰 숫자를 입력합니다
   
4. 사칙연산 기호를 입력합니다
   
5. 계산 결과를 확인합니다
    
6. 원하면 yes를 입력해 결과 삭제를 진행할 수 있습니다
    
7. exit를 입력하면 프로그램이 종료됩니다

## 파일 구조

- `App.java` 입력과 출력 담당
  
   - `Calculator.java` 계산 기능, 결과 저장, getter/setter, 삭제 기능을 담당
