## 주요 기능

- 두 개의 숫자 입력

- 사칙연산 기호 입력

- 덧셈, 뺄셈, 곱셈, 나눗셈 수행

- 나눗셈에서 0 입력 시 예외 처리

- 잘못된 연산자 입력 시 안내 메시지 출력

- 계산 결과를 ArrayList에 저장

- 저장된 결과 중 가장 먼저 저장된 값 삭제 가능

- Enum으로 연산자 타입 관리 (ADD, SUB, MUL, DIV)

- 제네릭으로 정수, 실수 모두 계산 가능

- 람다, 스트림으로 입력값보다 큰 결과값 조회

## 사용 기술

- Java

- Scanner

- ArrayList

- Enum

- Generic

- Lambda, Stream

## 실행 방법

1. 프로젝트를 실행합니다

2. 첫 번째 숫자를 입력합니다

3. 두 번째 숫자를 입력합니다

4. 사칙연산 기호를 입력합니다

5. 계산 결과를 확인합니다

6. 결과값보다 큰 값을 조회할 숫자를 입력합니다

7. 원하면 yes를 입력해 결과 삭제를 진행할 수 있습니다

8. exit를 입력하면 프로그램이 종료됩니다

## 파일 구조

- `App.java` 입력과 출력 담당

- `ArithmeticCalculator.java` 계산 기능, 결과 저장, 삭제, 조회 기능을 담당

- `OperatorType.java` 연산자 타입을 Enum으로 관리
